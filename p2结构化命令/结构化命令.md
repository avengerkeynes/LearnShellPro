## 结构化命令
	
#### if-then语句
```
	if command
	then
		command
	fi
```	
#### 不同于其他编程语言在if之后的对象是一个等式，这个等式结果为TRUE或FALSE,bash shell的if语句会运行if后面的命令，如果该命令退出状态码为0,则运行then如果不为0则then后不执行

#### 例 es1:
```
#!/bin/bash
#testing the if statement
if pwd
then
	echo "It worked"
fi
```
#### 例 es2:
```
#!/bin/bash
#testing a bad command
if IamNotaCommand
then
	echo "It worked"
fi
echo "We are outside the if statement"
```

#### test命令
#### test命令提供了if-then语句测试,如果test命令中列出条件成立,test命令便退出并返回状态码0,否则返回非零状态码,使得if-then语句不会再被执行
```
if test condition
then
	commands
fi

	bash shell 提供了另一种条件测试方法,无需声明test,
if [ condition ]
then
	commands
fi
```
#### 例 es6:

```
#!/bin/bash
```

#### 数值比较

比较|描述
--|:--


#### 文件比较
比较|描述
--|:--

#### 使用双括号
#### 双括号命令用来使用高级数学表达式

#### 双括号命令符号
符号|描述
--|:--
val++|后增
val--|后减
++val|先增
--val|先减
!|逻辑求反
~|位求反
**|幂运算
<<|左位移
>>|右位移
&|位布尔和
\||位布尔或
&&|逻辑和
\|\||逻辑或

#### 例 es23:
```
#!/bin/bash
#using double parenthesis
val1=10
if (($val1**2>90))
then
	((val2=$val1**2))
	echo "The square of $val1 is $val2"
fi
```
